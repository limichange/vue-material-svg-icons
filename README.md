## Install
```bash
$ npm i --save vue-material-svg-icons
# or
$ yarn add vue-material-svg-icons
```

## Usage
```js
import icon from 'vue-material-svg-icons/packages/icon-3d-rotation'
Vue.use(icon.name, icon)
```

## Development
```bash
# install dependencies
$ npm i
# or
$ yarn

# dev
$ npm run dev

# test
$ npm run test

# lint
$ npm run lint
```
